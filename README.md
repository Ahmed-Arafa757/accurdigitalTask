- Navbar contains:

- Main Body contains:
    - "Dashboard" title 
    - Main body contains:
        - Left side contains vertical navigation tabs
        - Right side contains:
            - List of chart types responsible for the visualization of data 
            - preview window to see the chart corresponding to the related data
